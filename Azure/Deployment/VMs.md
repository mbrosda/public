# Deployment of VMs

## Deploy VM from VHD

Follow the instructions in the Azure Quickstart Templates to deploy [generalized](https://github.com/Azure/azure-quickstart-templates/tree/master/201-vm-generalized-vhd-new-or-existing-vnet) VMs or [specialized](https://github.com/Azure/azure-quickstart-templates/tree/master/201-vm-specialized-vhd-new-or-existing-vnet) VMs from a VHD file to Azure.
